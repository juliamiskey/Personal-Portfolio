The Project

This project was the first responsive assignment during HackerYou's Web Development Immersive BootCamp. This project is meant to emulate a common task we will do as a front-end developer. We were provided a design and were tasked with building it out. We were required to use wrappers to constrain content on large displays, have a working navigation and use external .css files.

Toolkit

  • HTML5 /
  • CSS3
  • SASS /
  • Themeforest PSD files
  • Prepros

The Process

I began the process by creating a list of all the elements I would need. I used Adobe Assets to extract hex colours, the font-sizes of headers, paragraphs and to establish consistent spacing and margins. The project outline that we will not always be provided with the tablet/mobile views from a designer, and this project wass meant to put us in that situation. Part of being a front end dev can be making design/ux decisions - so we were asked to put some thought into how our users might interact with the site on various devices.

The Development

The development process for Simpleflex was very straight forward because the design was already laid out for us. I began by creating all elements that were shared between all three pages such as the navigation, buttons, headers and the footer. This made it easy to determine the structure going forward. I throughly enjoyed using BEM, CSS flexbox and grids.

The Result

Overall, the final product was almost identical to the PSD file and I am very proud of my work.
