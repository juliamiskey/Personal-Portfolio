The Project

This was my first time working with a partner. I had the pleasure of working with Li Ying Lim. As TV buffs, we decided to create an app that recommends TV shows based on a genre and year of choice.

The Process

The API we used was theMovieDatabase. It had great documentation and many endpoints. We worked on the project side-by-side to avoid any git merging conflicts. We had a lot of fun and we worked extremely well together.

The Development

Development Process went smoothly however, we did have to change some of our initial ideas due to limitations of the API. Instead of chosing a range of years, the user has to pick a specific year that would coorespond to the reslease date of the first season. We had to use dropdowns and a sub-dropdown according to what the user had selected from the first dropdown. This required many if-statements based on the value given.

The Result

Overall we attained all of our goals and requirements. If we were to go through the same project again, there are a few things we would do differently such as adding more results as well as ensuring the results were randomized so that way the users could see new results on every return.
