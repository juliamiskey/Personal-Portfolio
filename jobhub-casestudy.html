JobHub

Marketing Job Search App

View Live View on GitHub Home Page

The Project

For this project, we were asked to mimic an agency environment. We had to stick to a timeline, use only floats, and collaborate with fellow developers on Github. I was assigned to a group with Corey Lanier and Shatabdi Bhattacharjee. Our client, Ryan Christiani, asked that we make a job board for Marketing Jobs, using the Indeed API. He wanted the user to be able to set their location and be provided with a list of marketing jobs in their area. He also requested that the user would be able to save selected jobs and track information such as whether or not they have applied, as well as save notes.

The Process

We immediately agreed that we wanted strong branding and decided on a Logo, brand name and slogan. "JobHub: Search, Save, Succeed" was born. As a group, we began the process by wire-framing in which we decided to go with a mobile-first approach. We each had different strengths which we were able to capitalize on. Shatabdi was very familiar with Firebase, Corey was comfortable with React and I was confident in my design and API call skills. Beyond our primary skills, we then assigned different components to each other and with this, we felt much more organized and we were able to prioritize our tasks more efficiently.

The Development

Although this was our first time working React Router and a with a client, we were able to seamlessly execute our app. As expected, not everything worked out the way we intended, but this lead to problem solving that in turn create more wins to celebrate as a team.

The Result

Overall we were able to accomplish our goal and meet the need of our client. WE are happy with the results. If I were to go through the same project again, there are a few things I would probably do differently. For example, In the future, we plan on allowing the user to search for jobs beyond marketing. We plan to add a Job keyword input which in addition to the location. We also hope to allow the user to keep notes for individual saved jobs and get reminders to follow up with interviewers.
This project was an amazing way for me to learn more about React Router and the process of working with a client. It allowed me to and gain familiarity with the various tools needed and skills to work alongside a client with a clear concept in mind.
